## How to Use

1. Copy each prompt and paste into Midjourney
2. Wait for 4 variations to generate
3. Pick the best variation based on selection criteria (silhouette clarity, edge cleanliness, palette adherence)
4. Upscale your selection (U1-U4)
5. Save the upscaled image into the matching Art Book folder
6. Name it using the filename listed after each prompt

**Estimated total generation time: 3-4 hours**
